# 管理画面 機能モジュール（拡張用）

このフォルダには、今後のCRUD管理用機能を追加します。

## 予定している機能

| フォルダ名 | 内容 |
|-----------|------|
| `garbage/` | ゴミ出し（収集日・種別の管理） |
| `fortune/` | 占い（コンテンツ・結果の管理） |
| `game/`   | ゲーム（ミニゲーム設定・報酬の管理） |
| `receipt/`| レシート（キャンペーン・還元率の管理） |

## 追加の目安

- 各機能用の Firestore コレクション（またはサブコレクション）を定義する
- `features/<名前>/` に `screen.dart` と `service.dart` を置き、ダッシュボードからルートで遷移する
